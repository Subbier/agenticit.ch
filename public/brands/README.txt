Marken-Logos für die Excellence-Plattform-Wand
================================================

Legen Sie die offiziellen Logos hier ab — als SVG, jeweils quadratisch,
ca. 1 x 1 cm (z. B. 64 x 64 px viewBox). Dateiname exakt wie unten.
Solange eine Datei fehlt, zeigt die Kachel automatisch das Monogramm.

Hinweise:
- SVG wird bevorzugt (skaliert scharf). PNG geht auch — dann bitte kurz
  Bescheid geben, damit der Pfad von .svg auf .png umgestellt wird.
- Quadratisches Format mit etwas Innenabstand sorgt für ein ruhiges Raster.
- Original-Logos der Markeninhaber verwenden (Marken gehören den jeweiligen
  Eigentümern; Darstellung dient dem Hinweis auf Akkreditierung/Zusammenarbeit).
